## MODIFIED Requirements

### Requirement: 交互式地图渲染

系统 SHALL 使用 MapLibre GL JS 在浏览器中渲染交互式地图，底图由 Basemap Provider 架构根据 scale 自动选择。地图 MUST 采用古地图视觉体系：宣纸纹理背景、朱砂/靛蓝/赭石/墨色传统中国色配色、衬线中文字体。地图 MUST 根据当前页面模式采用不同的初始状态，在 timeline 模式下支持按事件步骤增量渲染。

地图 MUST 支持：
- 鼠标缩放（+/-）和拖拽平移
- 按 `source` 字段区分地名标记样式：CHGIS 精确地名→城池图标 ▭，LLM 推断地名→营寨图标 ▲
- **地名标记使用 zoom 表达式动态调整 icon-size**，远视角缩小、近视角放大
- **CHGIS 精确匹配的地名标记通过两个独立文字标签层分别显示古地名和今地名**
- 行军路线以**朱砂色（`#c23b22`）虚线 + 大箭镞符号**展示，模拟手绘行军线质感
- 每条路线可独立显示/隐藏
- 在 timeline 模式下，路线根据当前事件步数动态显示/隐藏
- 在 timeline 模式下，地名标记根据事件触发状态切换样式
- **部队以块状箭头展示**，宋军使用靛蓝（`#2b4c7e`），金军使用赭石（`#8b4513`）
- 部队箭头通过形态区分状态
- 部队箭头在 timeline 模式下根据当前 step 动态更新
- 多个部队关联同一地名时，箭头沿进攻方向平行错位避免重叠
- 箭头尺寸根据数据范围自适应
- 箭头形状为宽长比 1:3.5，头部占全长 40%
- 部队箭头图层可在地图图例区通过 checkbox 独立切换可见性

地图页面 MUST 使用衬线中文字体系列（KaiTi/Songti SC/Noto Serif SC/SimSun/Serif 回退栈）。

地图区域 MUST 添加卷轴式边框（`2px solid #2c2c2c` + 内阴影），地图背景 MUST 叠加 SVG noise 纹理模拟宣纸颗粒质感。

#### Scenario: 地名标记显示为城池/营寨图标

- **WHEN** 地图渲染地名数据
- **THEN** CHGIS 精确地名以双线方框城池图标显示，LLM 推断地名以三角营寨图标显示，不再出现实心圆点

#### Scenario: 行军路线为朱砂虚线

- **WHEN** 地图渲染行军路线
- **THEN** 路线以朱砂红虚线显示，带大箭镞方向符号，区别于现代导航实线样式

#### Scenario: 部队阵营使用传统中国色

- **WHEN** 地图渲染宋军和金军部队箭头
- **THEN** 宋军为靛蓝色（`#2b4c7e`），金军为赭石棕色（`#8b4513`），而非鲜蓝鲜红

#### Scenario: 地图文字使用衬线字体

- **WHEN** 页面加载完成
- **THEN** 所有地名标签、图例、UI 文字使用衬线中文字体（楷体/宋体系列）

#### Scenario: 地图背景有宣纸纹理

- **WHEN** 用户查看地图
- **THEN** 背景可见细腻的仿宣纸颗粒纹理，而非纯平面色块

#### Scenario: 地图有卷轴边框

- **WHEN** 用户查看地图
- **THEN** 地图区域外围有墨色细线边框和内阴影，呈现古地图卷轴感
