## Context

当前地图使用 OpenStreetMap raster 瓦片作为底图（`static/index.html` 第 256-264 行，inline style 定义）。所有战术数据层（地名、路线、部队箭头）叠加在 OSM 之上渲染。OSM 展示的现代街道/建筑与历史战役场景无关，形成视觉干扰。

## Goals / Non-Goals

**Goals:**
- 移除 OSM raster 瓦片依赖，替换为纯色背景
- 选择贴合历史地图氛围的背景色
- 改动最小化，仅触及 style 对象定义

**Non-Goals:**
- 不添加地形符号（Phase 2）
- 不改变任何数据层的颜色或样式
- 不引入新的渲染引擎或依赖

## Decisions

| 决策 | 选择 | 理由 |
|------|------|------|
| 背景实现方式 | MapLibre `background` 图层 | 原生支持，无需额外 source，性能最优 |
| 背景色 | `#f5f0e1`（仿古纸淡黄） | 贴近旧书页质感，历史氛围；与现有图层颜色（绿/黄/红/蓝）均有足够对比度 |
| 是否保留 OSM 引用 | 完全移除 | 无回退需求；如需恢复可从 git 历史找回 |

## Risks / Trade-offs

- 地名坐标仍按真实经纬度排布，但缺少地形参照物 → 后续 Phase 2（山川河流符号）解决
- 纯色背景下地图「空旷感」可能比预期更强 → 需实际使用后评估是否需要添加坐标网格或比例尺
